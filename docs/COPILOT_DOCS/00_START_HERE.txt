## 🎨 АРХИТЕКТУРА

```
POST /notes (создание заметки)
    ↓
Backend (REST API)
    ├→ Сохраняет в PostgreSQL
    └→ Отправляет tasks в Redis
        ├ extract:keywords
        └ compute:embedding
    ↓
Redis Queue
    ↓
Worker (async processor)
    ├→ Обрабатывает keywords
    │   └ Вызывает NLP service
    │   └ Сохраняет результаты
    └→ Обрабатывает embeddings
        └ Вызывает NLP service
        └ Сохраняет результаты
```
