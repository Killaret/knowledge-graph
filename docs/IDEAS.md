# Идеи для развития Knowledge Graph (не запланированы)

## Визуализация и интерфейс
- [ ] **Кластеризация графа / туман войны** – группировка звёзд по областям при отдалении, раскрытие при приближении (как Prezi).
- [ ] **Анимация появления узлов и связей** – плавное появление «звёзд» при загрузке графа.
- [ ] **Цветовая кодировка связей по весу** (толщина и цвет).
- [ ] **Горячие клавиши (hotkeys)** – быстрая навигация, создание заметок, сохранение.

## Интеграции
- [ ] **Синхронизация с календарями и сообщениями** – Google Calendar, Telegram, напоминания по встречам.
- [ ] **Импорт/экспорт заметок** (Markdown, JSON, Obsidian).
- [ ] **Парсер для загрузки текста** – автоматическое создание заметок и связей из большого текста (например, книги).

## Функциональность
- [ ] **Версионирование заметок** (история изменений, diff).
- [ ] **Разные коэффициенты для разных типов связей** (reference, dependency, related).
- [ ] **GraphQL** – для сложных агрегированных запросов (подсказки + рекомендации + граф одним запросом).
- [ ] **Очереди оповещений** – уведомления при новых связях/рекомендациях.
- [ ] **Сервис оркестрации (Saga)** – для распределённых транзакций (при росте).

## Мобильное и кроссплатформенное
- [ ] **Мобильное приложение** (React Native / Flutter или PWA) – доступ к базе знаний с телефона.
- [ ] **Адаптивный дизайн** – поддержка планшетов и мобильных браузеров.

## Прочее
- [ ] **Музыка и звуковые эффекты** – опциональная тема для атмосферы.
- [ ] **Поддержка русского и английского языков в NLP** (уже частично есть, можно расширить).
- [ ] **Отказоустойчивость и горизонтальное масштабирование** – реплики сервисов (при росте нагрузки).
